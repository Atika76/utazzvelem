- utazzvelunk.hu domain hivatkozások
- kijelentkezés utáni menü-visszaállás

FELTÖLTÉS

3. Várd meg, amíg a GitHub Pages újra deployol.
4. Domain aktiválás után Ctrl+F5 frissítés.
